### Подсказка по screen
запуск 
```bash
screen /dev/tty...
```
выход ctrl+a, затем d
вернуться обратно 
```bash
screen -r [ID]
```
ID смотрится в 
```bash
screen -ls
```
скриншот ctrl+a, затем h (сохранится там, где выполнен запуск)

завершить 
```bash
killall screen
```
